# 💖 UberLash

Um e-commerce desenvolvido para a **UberLash**, loja especializada em produtos para extensão de cílios.

O projeto foi desenvolvido com foco em uma experiência moderna, intuitiva e responsiva, proporcionando uma navegação agradável para clientes que desejam adquirir produtos para lash designers.

---

## 🚀 Tecnologias Utilizadas

* React
* Vite
* JavaScript (ES6+)
* Styled Components
* React Feather
* React Router
* HTML5
* CSS3

---

## ✨ Funcionalidades

* ✅ Página inicial responsiva
* ✅ Header com navegação
* ✅ Banner em formato de carrossel
* ✅ Listagem de produtos
* ✅ Cards de produtos
* ✅ Footer completo
* ✅ Exibição de formas de pagamento
* ✅ Exibição de formas de envio
* ✅ Layout moderno e organizado

---

## 📌 Funcionalidades em desenvolvimento

## ▶️ Como executar o projeto

Clone o repositório:

```bash
git clone https://github.com/gbc84389-tech/UberLash.git
```

Entre na pasta do projeto:

```bash
cd UberLash
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
